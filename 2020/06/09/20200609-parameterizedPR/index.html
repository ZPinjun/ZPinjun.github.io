Sparse Fourier Phase Retrieval Using Prony’s Method

2020-06-09

Abstract

The sparse prior can effectively improve the success probability of phase retrieval reconstruction. However, sparse phase retrieval algorithms such as GESPAR need to search for the support set, which leads to a long time to search for the correct support for large-scale signals. This letter proposes that for sparse Fourier phase retrieval, the support interval can be directly determined by the Prony’s method, and the possible support sets are determined into a small range through a simple reordering step. The simulation results prove that this is more efficient than the direct support search strategy. To complete phase retrieval, we also use a variant of Chebyshev’s method that proved to have a higher efficiency index than the damped Gauss-Newton method used in GESPAR algorithm to recover weights.

Introduction

Phase retrieval is a widespread inverse problem that focus on recovering a signal given the magnitude of its Fourier transform. It arises in various fields of science and engineering, including crystallography, radar waveform optimization, asronomy, and optical microscopy. Since the phase retrieval problem is ill-posed and NP-hard, there is no closed-form solution. Research in recent years has shown that sparsity prior can effectively improve the performance of phase retrieval algorithms. Shechtman et al. proposed a greedy algorithm called GESPAR for sparse phase retrieval in 2014. The GESPAR algorithm uses a two-stage strategy. It uses the 2-opt method to search for the support set, and then uses the damped Gauss-Newton method to optimize the amplitude values on each updated support set. he GESPAR algorithm has been successfully applied in many fields, such as X-ray coherent diffraction imaging and direction of arrival estimation. However, in addition to these encouraging results, the GESPAR algorithm also has an obvious flaw. That is because the support set is determined by search steps, it is impractical for large-scale signals.

This paper proposes a novel Fourier phase retrieval method with sparse prior. We pointed out that position intervals of the support set can be directly calculated by using Prony’s method, and then the possible support set can be determined into a small range through a simple reordering step.Then, we use a variant of the Cheyshev’s method that have a higher order of convergence than the damped Gauss-Newton method to recover the amplitude values.

Problem formulation

Using the notation $[n]_k:=\{k,k+1,\dots,k+n-1\}\subseteq\mathbb{N}$, $[n] = [n]_0$, and defining $\widehat{\mathbf{x}_N}$ is the $N$-point DFT result of signal $\mathbf{x}$, we consider the phase retrieval problem that recover $\mathbf{x}\in\mathbb{C}^M$ given $\mathbf{y}\in\mathbb{R}^N$, and these two signals satisfy
\begin{equation}
\mathbf{y}[n] = |\widehat{\mathbf{x}_N}[n]|^2, n\in[N],
\end{equation} where $M<N$.

We assume that $\mathbf{x}$ is a sparse signal of the form $\mathbf{x}[m]=\sum_{k=1}^{K}c_k\mathbf{h}[m-m_k], m\in[M]$, where $\mathbf{h}$ is a known basis function. In other words, the sparse bases are composed of a basis and its equally spaced displacements. If K is known beforehand, the original signal $\mathbf{x}$ can be uniquely determined by $2K$ parameters $\{c_i\}_{i=1}^K$ and $\{m_i\}_{i=1}^K$. Note $c_i\in\mathbb{R}$, $m_i\in[M]$, $\forall i\in[K]_1$. In this paper we aim to recover weights $\{c_i\}_{i=1}^K$ and support $\{m_i\}_{i=1}^K$ from the measurement $\mathbf{y}$.

Main proposed theorem

Defining map $\mathcal{B}$: $\{m_i\}_{i=1}^K \mapsto \{\Delta_{kl}|\Delta_{kl}=|m_k-m_l|,\forall k\neq l, k\in[K]_1,l\in[K]_1\}$, we have below Theorem.

Theorem 1: Consider a signal $\mathbf{x}\in\mathbb{C}^M$ in form of $\mathbf{x}[m]=\sum_{k=1}^{K}c_k\mathbf{h}[m-m_k], m\in[M]$, where the function $\mathbf{h}$ is known beforehand. Let set $Q = \mathcal{B}(\{m_i\}_{i=1}^K)$.\footnote{Due to the mutual difference property of sets, equal elements in $Q$ will merge into one element, so the size of $Q$ is less than or equal to $C_K^2$.} Assume $c_k\neq0,\forall k\in[K]_1$. Then set $Q$ can be uniquely characterized by $\mathbf{y}\in\mathbb{R}^N$ given by $\mathbf{y}[n] = |\widehat{\mathbf{x}_N}|^2$ if $N>2M$.
