斯坦福高效睡眠法
Created 2020-03-24| Updated 2020-03-25|读书
Post View:

最近在看西野精治写的一本书，《斯坦福高效睡眠法》。斯坦福大学睡眠研究所以及睡眠生物规律研究所（Sleep and Circadian Neurobiology Lab, SCN）被称为“世界第一睡眠研究所”，西野精治则是SCN研究所所长。这本书像是一本研究报告，介绍了一些睡眠实验以及新的结论。这篇文章记录我对其中一些要点的总结。

1. 睡得多不等于睡得好

所谓的最佳睡眠，“质”重于“量”。与睡眠相关的烦恼与精神压力，无法仅仅通过“对量的确保”来解决。

1.1 睡眠负债

睡眠时间不足，准确来说应称为“睡眠负债”。学术杂志Sleep，曾发表了有关睡眠负债的有趣实验。对刚值过夜班的医生和正常睡眠的医生进行反应能力测试，结果显示不睡觉的医生，大脑可能会出现将近4秒的无反应时间。在没有反应的时间，这些医生其实睡着了。这种状态被称为瞬时睡眠。瞬时睡眠指1秒~10秒的短暂的睡眠状态，它是大脑的一种防御机制。

瞬时睡眠出现的时候，当事人往往难以察觉。但是有时会造成很严重的后果，比如在开车的时候出现瞬时睡眠。

1.2 基因决定理想睡眠时间

有些物种天生不怎么需要睡觉。比如帝企鹅在孵小企鹅的1~2个月里，基本不睡觉；非洲有一种水牛，在发情期也可以做到好几周不睡觉。1965年，报纸记录了一位美国高中生挑战吉尼斯不眠记录的新闻，该高中生坚持了11天没有睡觉。整个过程使用脑电波仪来监测，所以可信度是比较高的。

在斯坦福大学也有这样的教授。当给他安装脑电波仪和活动检测仪后，发现他真的每天只睡4小时。即便如此，身体却十分健康，也没有给他的研究工作带来任何影响。西野精治本人也调研过一对美国父子，它们在数十年里，每天睡眠时间都不足6小时，也十分健康。调查发现，他们的遗传基因中与生物规律相关的时间遗传基因发生了变异。通过培养一只与这对父子拥有同样时间基因的老鼠，观察发现这只老鼠的睡眠时间果真也很短。

通过以上事实可以得出“基因决定理想睡眠时间”的结论。西野精治于2009年将这一结论发表在了学术期刊Science上。但是需要明确的是，绝大多数人都不具备短时间睡眠的基因。

基因决定了我们每天应该睡多长时间。虽然有的人天生只需要睡很短的时间，但是这也许并不是一件好事。有科学家通过药物让果蝇发生基因变异，观察发现，那些睡眠时间短的果蝇，寿命也很短；2002年，圣地亚哥大学的Daniel F.Kripke等人进行了一项100万人的跟踪调查，发现每天睡眠时间接近全体平均水平的人，死亡率最低。比平均水平过高或者过低的人群，其死亡率要高出1.3倍。对人类来说，收集大规模的准确数据是很困难的，但是从整体趋势来看，依然可以说睡眠时间短的人，寿命也相应更短一些。

此外，圣地亚哥大学的调查报告指出，睡眠时间短的女性，其肥胖度BMI值也较高。斯坦福大学、名古屋大学以及上海交通大学做过的关于死亡率和肥胖的研究结果，都与圣地亚哥大学的调查结果一致，这绝非偶然。

自2002年圣地亚哥大学公布研究报告以来，许多科学家意识到了睡眠的重要性，有许多新的结论被研究指出（部分）：

  • 不睡觉会导致胰岛素分泌紊乱，诱发糖尿病；
  • 不睡觉会导致增加食欲的胃饥饿素分泌，引发肥胖；
  • 不睡觉会导致交感神经长期紧张，引发高血压；
  • 存在睡眠负债的人，有易患痴呆症的可能；
  • 一天午休时间超过1小时，会加大痴呆症和糖尿病的风险。

1.3 偿还睡眠负债绝非易事

现代社会很多人都存在睡眠不足的问题。有人会提出可以利用周末补觉，补回睡眠负债。但事实上，这样做基本没有效果。

有一个实验让10个健康人随心所欲地睡觉。在实验前，它们的平均睡眠时间为7.5小时。在实验刚开始的时候，他们每天的睡眠时间都接近13个小时，而后逐渐下降，直到3周以后，他们的平均睡眠时间稳定在8.2小时上下波动。如果认为8.2小时是基因决定的理想睡眠时间，那么这群人在实验开始之前的很长一段时间里，每天都担负着40分钟的睡眠负债。而要还清这些负债，则需要3周以上的“随心所欲”的睡眠才能办到。

而另一个针对运动员的实验表明，还清睡眠负债后，他们的运动表现水平确实有所上升。

2. 睡觉时发生了什么

人进入睡眠后的状态在非REM睡眠和REM睡眠之间反复交替，且睡眠深度随时间递减，直到第二天早上醒来。入睡之后，最开始迎来的是非REM睡眠。特别是最初的90分钟，是睡眠全过程最深度的睡眠。通过对脑电波的监测，发现这一阶段会出现“大幅度且运动徐缓”的波形，大脑处于非活跃状态，也被称为徐波睡眠。入睡后不久出现的最深度睡眠90分钟，被称为黄金90分钟，是实现最佳睡眠的关键所在。在刚入睡时的黄金90分钟，如果能进入并保持深度睡眠，之后的睡眠就会变得很有规律且稳定，而且后面即使再被吵醒，对后续睡眠影响也不大。但如果在最初的90分钟阻碍睡眠，后续睡眠就会紊乱，甚至无法测量。所以，不建议叫醒刚睡着的人。

2.1 睡眠的五大使命

  1. 让大脑和身体都得到休息。尤其是在黄金90分钟内，副交感神经会占据主导地位，以此让大脑和身体得到休息，从而缓解压力；
  2. 整理并固化记忆。 在黄金90分钟内，信息从大脑的海马区转移到皮质中，而记忆也因此被保存下来了。而且优质的黄金90分钟睡眠，有助于帮助消除糟糕的记忆；
  3. 调节激素平衡。尤其是生长激素，在黄金90分钟内会大量分泌。成年人也需要依靠这种激素，才能让肌肉和骨骼变得健壮，保持新陈代谢；
  4. 提高免疫力 实验证明，睡眠不好，免疫力也会出现异常；
  5. 排除大脑中的废物 人的脑脊液容量大概有1500毫升。我们的身体每天会分四次替换掉其中600毫升。当我们处于清醒时，神经细胞活跃，大脑中的废物会不断堆积。所以晚上大脑会进行自身的“维护保养”工作。如果大脑的废弃物不能及时排出，就有可能成为阿尔茨海默症的诱因。

2.2 睡眠终点站——梦

在1957年，斯坦福大学的德门教授就指出，非REM睡眠状态下人也会做梦，之后有数名研究人员对此结论做了进一步的证实。起床后记得的梦，通常是不久前做的梦。REM睡眠时所做的梦，大多具有故事性，并且具有个人真实体验性；而非REM睡眠时做的梦，则大多是抽象的，并且不太符合逻辑。在正常的睡眠节奏下，普通人会做七八次不同的梦。遗憾的是，睡眠越充分，我们越是只记得最后一个梦。REM睡眠被提出后，关于“人是否可以做指定内容的梦”的验证研究一度很流行。但大量的实验表明，人不可能控制梦的内容，除了偶然巧合外。
